# python-selenium-grid-docker
Simple Python (pytest) based selenium grid 4 docker test run setup

## run tests on local 
```shell
./run_tests.sh
```

## run tests on docker 
```shell
./run_tests_in_docker.sh
```

## initial setup (optional)
```shell
python -m pip install virtualenv
virtualenv venv
source ./venv/bin/activate
```
